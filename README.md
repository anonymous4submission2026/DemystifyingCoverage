# DemystifyingCoverage

This is the artifact repository of the paper titled "Demystifying Coverage-Guided Compiler Fuzzing". The repository is structured by the research questions as `rq1`, `rq2`, and `rq3`. Data and script files are inside the corresponding `rq` directory.

Scripts used to run fuzzing experiments and to organize crashes are present inside the `rq3` directory. Additional plotting scripts can be found in the respective `rq` directories.

We will submit an archived version of the artifacts upon acceptance.
